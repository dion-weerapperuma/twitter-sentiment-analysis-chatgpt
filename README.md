# Twitter Sentiment Analysis — ChatGPT Dataset

## Overview
Analysed 50,000 tweets about ChatGPT to extract sentiment 
trends, identify top hashtags and active users, and model 
discussion topics across OpenAI, Google and Microsoft mentions.

## Tools & Libraries
- Python (Pandas, NLTK, Scikit-learn, Matplotlib)
- Sentiment Analysis using VADER
- Topic Modelling using LDA
- TF-IDF Vectorisation

## Key Findings
- #AI was the most mentioned hashtag with 2,610 mentions
- Sentiment analysis performed across OpenAI, Google 
  and Microsoft tweet mentions
- LDA topic modelling identified 10 distinct discussion themes

## Dataset
This project uses the ChatGPT Twitter dataset.
Download from Kaggle and upload to your Google Drive 
before running the notebook.

## Setup
1. Upload chatGPT-Twitter.csv to your Google Drive
2. Open analysis.ipynb in Google Colab
3. Run all cells from top to bottom
